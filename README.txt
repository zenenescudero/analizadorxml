Desarrollo de aplicacion en vb.net  tipo cliente servidor.


utiliza base de datos mysql 5.0 

Descripcion:

sistema para el analisis masivo de archivos xml CFDI.

obteniendo de cada archivo los datos relevantes como son.

* RFC emisor y receptor
* Razon Social de Emisor y Receptor.
* Fecha de Timbrado.
* Folio Fiscal.
* Serie y Folio.
* Importe
* No. de Recibo y De Orden de Compra (Parte Addenda).

generando un reporte en excel para estos datos.


estos datos se pueden guardar en la base de datos y cargar informacion de las tablas de ERP SAP para hacer un cruzado de informacion mediante

* RFC y FOLIO

* No de Recibo y Ordenes de Compra.

* UUID
